# Public Site Package

This folder is the public static export for the lite viewer.

## Start locally

```bash
python3 -m http.server 4173
```

Then open:

- [http://localhost:4173](http://localhost:4173)

## Suggested deployment targets

- Cloudflare Pages
- Netlify
- Vercel static hosting
- GitHub Pages

Deploy this folder as a static site root.
